## delete

    delete {range}

range = comma separated range of row indices (0 based) e.g.

    0           # delete row 0
    1,2,5       # delete row 1,2,5 
    1:5,10:15   # delete row 1 to 5 and row 10 to 15
    1,3,5:10    # delete row 1,3 and 5 to 10

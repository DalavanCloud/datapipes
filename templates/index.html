Data Pipes

This is a pre-alpha demonstrator – check out the code and issues on github …

Streaming Online Data Transformation!

Data Pipes is an online service to provide unix-style cut, grep, sed in a streaming, "pipe-like" manner on the web (you can also run it locally!).

We need data that streams so, at the moment, we only operate on CSV files.

Examples

Delete the rows 1-3 of a CSV file:
/csv/delete?range=1:4&url=http://static.london.gov.uk/gla/expenditure/docs/2012-13-P12-250.csv
